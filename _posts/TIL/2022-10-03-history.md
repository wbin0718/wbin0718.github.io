# 딥러닝의 역사

## 인공지능이란?

> 사람의 지능을 모방하는 것.

- 인공지능 > 머신러닝 > 딥러닝
- 딥러닝의 요소
    1. data : 풀고자하는 문제 의존적 ex) 분류, segmentation, detection, pose estimation 등
    2. model : 모델의 구조는 다름.
    3. loss function : 모델을 어떻게 학습할지. ex) MSE, CE, MLE
    4. 알고리즘 : 네트워크를 어떻게 줄일지.
